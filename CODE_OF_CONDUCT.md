# Code of Conduct

Nous nous engageons à rendre la participation à ce projet accueillante et sûre pour tous.

## Règles

- Soyez respectueux envers tous les contributeurs
- Pas de harcèlement, de discrimination ou de propos offensants
- Signalez tout comportement inapproprié aux maintainers

## Contact

Pour toute question ou signalement, contactez les auteurs via le dépôt GitHub.
